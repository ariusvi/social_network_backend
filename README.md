## Local installation ğŸ’»
1. Clon repository    
 ` $ npm init --yes `
2. Instal express    
 ``` $ npm i express ``` 
3. Instal nodemon   
 ``` $ npm i nodemon -D ``` 
 ``` $ npm i dotenv -E```
4. Add type module into package.json    
```json
{
	"type": "module"
}
```
5. Instal mongoose   
 ``` $ npm i mongoose ```
6. Instal bcrypt    
 ``` $ npm i bcrypt ```

``` $ npm i jsonwebtoken ```
